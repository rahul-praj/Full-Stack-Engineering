CSS Selectors and Specificity

This is a cheatsheet to basic CSS selectors that can be referenced to style HTML sheets.

Property Description Specificity
#id The id property is used to reference items in the HTML code that have been assigned an ID. To reference ID's, it needs to be preceded with a hashtag in the CSS file. High
.class The class property is used to reference classes set in the HTML code. To reference classes, the class name needs to be preceded with a . in CSS Medium
Types You can reference elements such as p or div directly. Low
Descendant combinators This refers to selecting elements nested within other elements, known as descendants. This has a higher degree of specificity than singular elements on their own. High

Types

This is the types section

The declaration is important for specifying a style to apply, but in order to specify which text type (e.g. paragraphs, headings) to apply that style to, you need a selector. Types describe anything created by a tag. For example, p, div, body are all examples of types.

Descendant Combinators

This is the section detailing descendant combinators

CSS also supports selecting elements that are nested within other HTML elements, known as descendants. Note that with descendant combinators, specificity increases.
